# Asset Management System

A Python + MySQL based Asset Management System with authentication and department-wise asset management.

## Features

- Admin login
- Developer login
- User access
- Add assets
- Update assets
- Delete assets
- Display assets
- Password hashing using bcrypt
- Department-wise asset management

## Technologies Used

- Python
- MySQL
- PyMySQL
- bcrypt

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the program:

```bash
python main.py
```

## Database

The project automatically creates:
- Database
- Tables
- User system
